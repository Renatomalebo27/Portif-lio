# Portfólio (template)

Abra o arquivo `index.html` no navegador para ver o site.

Estrutura:
- `index.html` — página principal do portfólio
- `styles.css` — estilos responsivos
- `script.js` — interações simples (ano, smooth scroll, envio básico)

Como usar:
1. Substitua "Seu Nome" e o conteúdo dos projetos.
2. Atualize o email em `script.js` para receber mensagens.
3. Abra `index.html` no navegador (duplo clique) para visualizar.

Quer que eu personalize cores, fontes ou adicione páginas individuais para projetos? Peça que eu faça as alterações.
